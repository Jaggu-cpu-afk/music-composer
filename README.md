# 🎧 Mood-Based Music Player

A smart web application that detects user mood through text input and plays music accordingly.

---

## 🌐 Live Demo

👉 [🚀 Click Here to Open Website](https://jaggu-cpu-afk.github.io/music-/)

---

## ✨ Features

- 💬 Chatbot-based interaction
- 🎧 Mood detection system
- 🎵 Music player integration
- 🖼️ Dynamic album art change
- 🎶 Animated equalizer (only when playing)
- ❤️ Favorites system
- 📊 Mood history tracking
- 🌐 Deployed using GitHub Pages
